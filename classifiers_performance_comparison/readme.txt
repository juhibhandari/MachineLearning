Question:
Performance comparison of following classifiers available in ML: Logistic Regression, k-Nearest Neighbors, Bagging, Random Forests, AdaBoost


Step 1: Install these packages
Packages : stats,class, ROCR, party, ipred, ada, randomForest

Step 2: Read the dataset from the link : https://archive.ics.uci.edu/ml/machine-learning-databases/ionosphere/ionosphere.data 

Step 3: Run the code. 
The results get displayed on the console. 

Evaluation metric = ROC curve plots. 

Classifiers analysed:
---------------------
• Logistic Regression
• k-NN
• Bagging
• Random Forest
• Boosting

List of files:
----------------
log_regression.r
knn.r
bagging.r
random_forest.r
boosting.r

result.doc - ROC plots, results, pseudocode and analysis
logfile.txt - accuracy of each classifier for different folds to find the best parameter for each model
